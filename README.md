# 📄 Cargador de Facturas - Monotributo Argentina

Aplicación para automatizar la carga de facturas C y notas de crédito C en Excel/Google Sheets.

## ✨ Características

- **Extracción automática** de datos desde PDFs de facturas
- **Interfaz gráfica intuitiva** fácil de usar
- **Exportación a Excel** (.xlsx)
- **Soporte para múltiples archivos** simultáneos
- **Registro de operaciones** (logs)
- **Barra de progreso** en tiempo real

## 📋 Requisitos

- Python 3.8 o superior
- Librerías:
  - `PyPDF2`
  - `pandas`
  - `openpyxl`
  - `tkinter` (incluido en Python)
